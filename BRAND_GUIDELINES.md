# Brand Guidelines

The project name, logo, and associated branding are not granted under the open-source or content licenses unless explicitly stated.

## Allowed
- Forking and modifying the code/content under the respective licenses
- Attribution stating that a project is "based on" or "derived from" this project

## Not Allowed (without written permission)
- Using the project name or logo in a way that implies official status or endorsement
- Marketing a fork as the official version
- Reusing the logo or visual identity as-is for a public fork

## Recommended attribution
“Based on the original project by the original author (link to the original repository).”
